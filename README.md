CovoitApp est une application Laravel permettant de gérer un système de covoiturage destiné aux employés effectuant des déplacements entre différents campus.

Cette application suit une architecture MVC, propre à Laravel, et met en place les éléments suivants : migrations, modèles Eloquent, factories, seeders, contrôleurs, vues et relations entre les différentes entités du système.

L'application permet de gérer les employés, leurs véhicules, ainsi que les trajets proposés entre les campus, en précisant le conducteur, le véhicule utilisé et les campus de départ et d’arrivée.

Les employés peuvent fréquenter plusieurs campus et s’inscrire comme passagers à des trajets, avec une gestion du nombre de places disponibles.

Cette application garantit une organisation structurée des déplacements, favorisant le partage de véhicules et l’optimisation des trajets inter-campus.
